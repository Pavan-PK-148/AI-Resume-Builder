## ✨ Features

### 🤖 AI-Powered Resume Enhancement
- Generate and improve resume content using OpenAI  
- Smart rewriting for better clarity and impact  
- ATS-friendly formatting suggestions  

### ⚡ Core Functionalities
- Real-time resume editing with live preview  
- Create, update, and manage multiple resumes  
- Clean and responsive UI for better user experience  

### 📊 Smart ATS Scanner
- Analyze resume against job descriptions  
- Get an ATS compatibility score (e.g., 85% → 92%)  
- Identify missing keywords and skill gaps  
- Actionable improvement suggestions  

### 📄 PDF Upload & Parsing
- Upload existing resume in PDF format  
- Automatically extract and structure content  
- Convert old resumes into editable formats instantly  

### 🔍 Keyword Optimization Engine
- Suggest high-impact keywords based on job roles  
- Improve resume ranking in ATS systems  
- Highlight missing skills and technologies  
- Dynamic keyword tagging system  

### 🔐 Authentication & Security
- JWT-based login and signup system  
- Secure API communication using Axios interceptors  
- Protected routes and user sessions  

### 🌍 Resume Management & Sharing
- Store multiple resumes using MongoDB  
- Edit and manage resumes anytime  
- Generate public shareable resume links  

### 🖼️ Image Integration
- Upload profile images using ImageKit  
- Optimized storage and fast delivery  

---

## 🛠️ Tech Stack

### 🎨 Frontend
- React.js (Vite)  
- Tailwind CSS  
- Axios (with interceptors)  
- Lucide React Icons  

### ⚙️ Backend
- Node.js  
- Express.js  
- MongoDB (Mongoose)  
- OpenAI API  
- ImageKit SDK  

### ☁️ Deployment
- Frontend: Netlify  
- Backend: Render  
- Database: MongoDB Atlas

## ⚙️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Pavan-PK-148/AI-Resume-Builder.git](https://github.com/Pavan-PK-148/AI-Resume-Builder.git)
Backend Setup:

Navigate to /server

Run npm install

Create a .env file with: MONGODB_URI, JWT_SECRET, OPENAI_API_KEY, IMAGEKIT_PRIVATE_KEY

Run npm start

Frontend Setup:

Navigate to /client

Run npm install

Create a .env file with: VITE_API_URL=http://localhost:3000

Run npm run dev
